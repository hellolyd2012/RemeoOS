# RemeoOS

## 简介

![RemeoOSLogo](Docs/Img/Logo.png)

RemeoOS 是一个不基于任何现有操作系统的操作系统

## 编译

#### 配置环境

首先确保电脑上安装了一下软件

- GCC
- G++
- QEMU
- BOCHS
- MAKE
- GDB

#### 编译

在项目根目录下执行 `make all` 即可编译

#### 运行

在项目根目录下执行 `make run` 即可运行

#### 调式

##### QEMU

在项目根目录下执行 `make debug-qemu` 即可进入 QEMU 调试模式

##### BOCHS

在项目根目录下执行 `make debug-bochs` 即可进入 BOCHS 调试模式
